![Screenshot](http://i.imgur.com/gy8YLjL.png)


## Installation

Use stack. Really. If you don't want to use stack, you're on your own for now.

```
# Arch: aura -Akax haskell-stack
# Otherwise: cabal install -j stack
stack setup
stack test
stack install
```
